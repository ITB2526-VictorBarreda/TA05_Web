Project Overview

Hands-on networking fundamentals

This project explores the core concepts of computer networking, including IP addressing, subnetting, routing, and essential services like DNS and DHCP. Using tools such as Cisco Packet Tracer and Wireshark, I simulated real-world network topologies and analyzed packet flows to understand how data travels across networks.

  • Tools used: Cisco Packet Tracer, Wireshark, GNS3
  • Skills developed: Subnetting, static/dynamic routing, NAT, firewall rules
  • Protocols: TCP/IP, ICMP, ARP, DNS, DHCP

Through this project, I gained a solid foundation in network design and troubleshooting, preparing me for real-world IT infrastructure challenges.
