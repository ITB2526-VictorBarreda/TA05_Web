Project Overview

Building efficient and accessible websites

This project focused on creating a sustainable and accessible website using semantic HTML and modern CSS. I prioritized performance, low energy consumption, and inclusive design to ensure the site was usable by everyone and environmentally conscious.

  • Tools used: VS Code, Lighthouse, GitHub Pages
  • Skills developed: Semantic HTML, responsive design, accessibility (WCAG), performance optimization
  • Highlights: Lazy loading, reduced DOM complexity, color contrast compliance

The result was a fast-loading, mobile-friendly portfolio that aligns with digital sustainability principles and offers a great user experience across devices.
