Networking & Services

Infrastructure, protocols, and connectivity

Brief: Learn how networks are built, configured, and secured using real-world tools and protocols.

Details: This project explores the fundamentals of computer networking, including IP addressing, subnetting, VLANs, routing protocols (OSPF, RIP), and essential services like DNS and DHCP. You'll configure firewalls using iptables and pfSense, analyze traffic with Wireshark, and simulate topologies in Packet Tracer. The goal is to build a secure, scalable, and efficient network infrastructure from scratch.

Web Development & Sustainability

Markup, accessibility, and green UX

Brief: Build fast, accessible, and eco-friendly websites using semantic HTML and responsive design.

Details: This project focuses on creating sustainable web experiences. You'll learn to structure content with semantic HTML5, apply responsive layouts with CSS Flexbox and media queries, and optimize performance for low energy consumption. Accessibility is a core focus, using ARIA roles and contrast-aware design. You'll also audit your site with Lighthouse and the Website Carbon Calculator to reduce digital emissions.
